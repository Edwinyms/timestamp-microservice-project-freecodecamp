Timestamp Microservice

To test this app, simply click on the urls below and ready to be astounded by this apps capability to convert unix dates to UTC and vice versa.

Else, if you want to try a custom date, simply enter the date in the adress bar.

For example, if you wanted to enter a unix date, add the following text to the end of the existing url (/api/timestamp/1451001600000)

For example, if you wanted to enter a natural date, add the following text to the end of the existing url (/api/timestamp/2016-12-16)

Example Usage:

  • https://timestamp-microservice-project-freecodecamp.glitch.me/api/timestamp/2015-12-25
  • https://timestamp-microservice-project-freecodecamp.glitch.me/api/timestamp/1450137600

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
